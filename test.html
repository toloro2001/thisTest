This terminal can't be displayed because you are accessing the game outside of the recommended virtual machine. To download the virtual machine please follow the instructions below. Alternatively you can use your own Terminal, such as PuTTY on Windows or the inbuilt terminal on Mac OS X. We strongly recommend using the virtual machine for maximum compatibility.

Windows VMWare Instructions:

  • Download VMWare Player by clicking here.
  • Double click on the file to install it.
  • Once installation has completed, run VMWare Player.
  • Download the VMWare Image by clicking here
  • Then extract the downloaded image using WinZip / WinRAR / 7zip.
  • And in VMWare Player go to File -> Open and navigate to the location you extracted the image to and open the .vmx file.
  • Once the Virtual Machine starts up, the login details are username: agent and password: agent

Mac OS X VirtualBox Instructions:

  • Download Virtualbox by clicking here.
  • Double click on the dmg, then double click on VirtualBox.pkg.
  • Once installation has completed, find Virtualbox in your applications folder and run it.
  • Download the Virtualbox Image by clicking here
  • Then extract the downloaded image.
  • In Virtualbox go to the menu bar at the top, and Machine -> Add and navigate to the Virtual Machine location which you extracted.
  • Once the Virtual Machine starts up, the login details are username: agent and password: agent
